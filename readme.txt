System Resource Monitor by Nick Conn
--------------------------------------------
- This project consists of a simple GUI that displays the system's available resources
- It displays the CPU usage, RAM usage, and the amount of storage space available
- The CPU and RAM usage is logged in a text file (systemresourcelog.txt) every 2 seconds along with a timestamp
- There are also real time graphs using matplotlib to display the cpu and ram usage

This project was made with python and utilizes the following libraries: tkinter, psutil, matplotlib

In order to run this project you will need python installed as well as these libraries.
You can install them by running the following pip commands:

- pip install tkinter
- pip install matplotlib
- pip install psutil

Once all libraries are installed, you can run main.py to start the program.
